# ASAProxies

### Front-end Requirements:

- npm

### Back-end Requirements:

- python3
- pip

### Front-end Setup:

- cd frontend
- npm install
- npm start

### Back-end Setup:

- cd backend
- python3 -m venv env
- pip install django
- pip install djangorestframework
- pip install django-cors-headers
- pip install django-allauth
- pip install django-rest-auth
